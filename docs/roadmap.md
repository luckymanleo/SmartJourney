## v2.2 ✅ — 性能优化（2026-06 已完成）

## Phase 6 — 体验深化（计划中）

### 混合交通智能组合 (P0)
- 自动生成"飞机+高铁+打车"联程方案
- 比价矩阵：直飞 vs 高铁 vs 中转组合
- 换乘时间衔接优化

### 多段联程 (P0)
- 多城市旅游自动编排（"上海→杭州→苏州 7天"）
- 城市间交通自动衔接
- 环线行程智能推荐

### 实时预警 (P0)
- 航班延误/列车停运推送通知
- 目的地天气突变预警

### 出行前 Checklist (P1)
- 自动生成出行准备清单（证件、衣物、药品、充电器等）
- 基于目的地+天数+季节智能推荐
- 勾选确认进度

### 行中助手 (P1)
- 值机提醒（提前 24h / 2h）
- 下一段行程倒计时提醒
- 站台/登机口变更通知

### 补充出行方式 (P1)
- 打车/网约车查询与比价
- 长途大巴/城际客运班次查询
- 租车/自驾服务

### 统一票夹 (P1)
- 跨平台订单聚合展示
- 行程单 PDF 生成
- 一键导出报销材料包

### 退改签政策聚合 (P2)
- 不同票种退改签规则可视化对比
- 灵活性评分（1-5 星）

### 国际化 (P2)
- 多语言界面（英文优先）
- 境外数据源接入（Booking.com / Skyscanner）
- 多币种 + 实时汇率

## 长期愿景

### 生态开放
- **MCP 工具市场**：允许第三方开发者贡献新的旅行 MCP Server
- **开放 API**：允许其他 App 嵌入 SmartJourney 规划能力

### AI 进阶
- **多模态交互**：拍照识别景点/菜单/路牌
- **语音规划**：语音输入出行需求
- **AR 导航**：景点/机场/火车站 AR 实景导航
- **行程复盘**：旅行后 AI 自动生成精美游记 + 数据总结

### 商业闭环
- **会员体系**：积分、等级、专属优惠
- **企业版**：差旅管理（TMC）解决方案

## 技术债与架构演进

| 项目 | 当前 | 未来演进 |
|------|------|----------|
| 数据库 | PostgreSQL 单机 | 读写分离 / 分片 |
| 缓存 | Redis 单机 | Redis Cluster |
| MCP 通信 | HTTP (ModelScope) | 多数据源 fallback |
| 搜索 | 实时查询 | Elasticsearch 全文索引 |
| 消息推送 | SSE 流式推送 | RabbitMQ/Kafka + WebSocket |
| 文件存储 | 本地磁盘 | OSS（阿里云/腾讯云） |
| CI/CD | 手动部署 | GitHub Actions 自动构建 |
| 监控 | 基础日志 | Prometheus + Grafana |
| APM | 无 | Sentry 错误追踪 |
