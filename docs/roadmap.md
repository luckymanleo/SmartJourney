## 未来计划 — 按优先级排序

### P0 — 近期（直接提升核心体验）

| # | 功能 | 说明 | 改动量 |
|---|------|------|--------|
| 1 | **MCP 多数据源 fallback** | ModelScope 频繁超时，接入备用 MCP 或直连云厂商 API | 后端 |
| 2 | **AI 规划容错增强** | MCP 部分失败时仍生成行程 | 后端 |
| 3 | **行程导出** | 行程单 PDF / 图片分享 | 后端+前端 |
| 4 | **端到端测试** | 覆盖登录→搜索→规划→保存完整链路 | 测试 |
| 5 | **🗺️ 搜索结果地图** | 酒店/景点/美食搜索结果标记在地图上，左右分栏联动 | 前端 |
| 6 | **🗺️ AI 规划实时地图** | SSE 规划流式返回时，POI 逐一亮起，完成后即完整路线图 | 前端 |

## 技术债（按需偿还）

| 项目 | 当前 | 演进方向 |
|------|------|----------|
| MCP 通信 | ModelScope HTTP | 多数据源 + fallback |
| 日志 | 按日切分文件 | 结构化日志 + 集中收集 |
| 测试 | 手工验证 | CI + e2e |
| 监控 | 基础日志 | Prometheus + Grafana |
| 错误追踪 | 本地 error.log | Sentry |
| 部署 | 手动 | Docker Compose 一键部署 |
| 数据库 | PostgreSQL 单机 | 读写分离 |
| 缓存 | Redis 单机 | Redis Cluster |
