## P0 — 稳定性（当前版本缺陷修复）

| # | 功能 | 说明 | 改动量 |
|---|------|------|--------|
| 1 | **MCP 多数据源 fallback** | ModelScope 频繁超时/空结果，接入备用 MCP 或直连云厂商 API | 后端 |
| 2 | **AI 规划容错增强** | MCP 部分失败时仍生成行程，降级使用常识补全缺失数据 | 后端 |

## 技术债

| 项目 | 当前 | 演进方向 | 优先级 |
|------|------|----------|--------|
| MCP 通信 | ModelScope HTTP 单点 | 多数据源 + 自动 fallback | P0 |
| 测试 | 手工验证 | CI + e2e + 单元测试覆盖 | P0 |
| 日志 | 按日切分文件 | 结构化 JSON 日志 + 集中收集(ELK) | P1 |
| 部署 | 手动启动 | Docker Compose 一键部署 + healthcheck | P1 |
| 监控 | 基础日志 | Prometheus + Grafana 面板 | P2 |
| 错误追踪 | 本地 error.log | Sentry 实时告警 | P2 |
| 数据库 | PostgreSQL 单机 | 读写分离 + 连接池监控 | P2 |
| 缓存 | Redis 单机 | Redis Cluster / Sentinel 高可用 | P3 |
