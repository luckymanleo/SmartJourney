## Phase 6 — 体验深化（计划中）

### 混合交通智能组合 (P0)
- 自动生成"飞机+高铁+打车"联程方案
- 比价矩阵：直飞 vs 高铁 vs 中转组合
- 换乘时间衔接优化（最小转机时间建议）
- 中转城市短暂停留推荐

### 多段联程 (P0)
- 多城市旅游自动编排（"上海→杭州→苏州 7天"）
- 城市间交通自动衔接
- 环线行程智能推荐

### 退改签政策聚合 (P1)
- 不同票种/舱位退改签规则可视化对比
- 灵活性评分（1-5星）
- "灵活票"推荐（行程不确定时的最佳选择）

### 统一票夹 (P1)
- 跨平台订单聚合展示（飞猪+美团+手动添加）
- 行程单 PDF 生成（适合报销）
- 一键导出报销材料包（行程单+发票提醒）

### 实时预警 (P0)
- 航班延误推送通知
- 列车停运/晚点提醒
- 目的地天气突变预警
- 动态出发时间调整建议

### 行中助手 (P1)
- 实时消息推送（微信服务通知 / 短信）
- 值机提醒（提前 24h / 2h）
- 站台/登机口变更通知
- 下一段行程倒计时提醒

### 补充出行方式 (P1)
- 打车/网约车查询与比价（滴滴/高德/美团）
- 长途大巴/城际客运班次查询
- 景区专线查询
- 租车/自驾（取还车点、车型、保险）

### 技术增强
- PWA 离线缓存（搜索结果、已保存的行程）
- 地图导航跳转（高德/百度地图 Deeplink）
- 性能优化（首屏加载 < 1.5s）

## Phase 7 — 社交深度（计划中）

### 多人出行协同 (P0)
- 行程组队创建（创建者 + 成员）
- 邀请机制（分享链接/扫码加入）
- 成员角色（发起人/编辑者/查看者）
- 实时位置共享（地图上显示成员位置）
- 行程变更通知所有成员

### 账单分账 (P0)
- 共同消费记录（某人垫付 → 自动分账）
- AA 分账计算（等额/按比例/自定义）
- 结算汇总与提醒
- 多币种支持（国际出行）

### 行程分享 (P1)
- 行程海报生成（图片，适合分享朋友圈）
- 行程公开链接（只读分享）
- 游记自动生成（基于实际行程 + AI 润色 + 照片）

### 餐饮深度服务 (P1)
- 在线排队取号（接入美团排队 API）
- 餐厅预订确认
- 沿途餐饮智能推荐（根据行程路线）

### 行李服务 (P2)
- 行李寄存点查询（机场/火车站/热门商圈）
- 机场→酒店行李寄送服务查询
- 超规行李提醒

### 旅行保险 (P1)
- 根据行程天数/目的地/活动类型推荐保险
- 保险方案对比（保障范围+价格）
- 一键投保跳转

### 接送机服务 (P1)
- 接送机专车查询与比价
- 航班延误自动调整接机时间

## Phase 8 — 智能化与闭环（持续迭代）

### AI 深度偏好学习 (P0)
- 基于历史行程的偏好建模
- 主动推荐：常去目的地促销、偏好航司特价
- 个性化排序（搜索结果按用户偏好排序）
- "猜你喜欢"推荐流

### 异常处理与自动改签 (P0)
- 航班取消后自动搜索替代方案（前后2天+周边机场）
- 智能改签推荐（最小时间损失+最小费用）
- 一键改签跳转
- 航班超售/延误的权益提醒

### 无障碍出行 (P2)
- 轮椅服务查询与预约
- 特殊协助需求标注（行动不便/视听障碍/老人/孕妇）
- 无障碍酒店/景点/交通筛选

### 出入境信息 (P1)
- 签证要求查询（中国护照 → 目的地）
- 签证办理入口
- 出入境注意事项（海关限制/防疫要求）
- 实时汇率 + 换算器
- 常用语翻译卡片

### 公共交通深度集成 (P1)
- 公交线路查询
- 地铁线路图
- 乘车码跳转（支付宝/微信）
- 共享单车扫码跳转
- 步行导航

### 渡轮/游船 (P2)
- 沿海轮渡班次查询（如厦门→鼓浪屿、大连→烟台）
- 内河游船查询
- 海岛航线

### 国际化 (P2)
- 多语言界面（英文优先）
- 境外出行支持（国际机票/酒店/活动）
- 境外数据源接入（Booking.com/Expedia/Skyscanner MCP）

## 长期愿景

### 生态开放
- **MCP 工具市场**：允许第三方开发者贡献新的旅行 MCP Server
- **插件系统**：让酒店/航司/景区上传自己的数据源
- **开放 API**：允许其他 App 嵌入 SmartJourney 规划能力

### AI 进阶
- **多模态交互**：拍照识别景点/菜单/路牌，自动关联行程
- **语音规划**：语音输入出行需求，全程语音交互
- **AR 导航**：景点/机场/火车站 AR 实景导航
- **行程复盘**：旅行后 AI 自动生成精美的游记 + 数据总结

### 商业闭环
- **会员体系**：积分、等级、专属优惠
- **智能推荐广告**：基于行程的精准酒店/租车/保险推荐（非侵入式）
- **企业版**：差旅管理（TMC）解决方案

## 技术债与架构演进

| 项目 | 当前 (Phase 1-5) | 未来演进 |
|------|-----------------|----------|
| 数据库 | PostgreSQL 单机 | 读写分离 / 分片 |
| 缓存 | Redis 单机 | Redis Cluster |
| MCP 通信 | HTTP (ModelScope) | 多数据源 fallback |
| 搜索 | 实时查询 | Elasticsearch 全文索引 |
| 消息推送 | SSE 流式推送 | RabbitMQ/Kafka + WebSocket |
| 文件存储 | 本地磁盘 | OSS（阿里云/腾讯云） |
| CI/CD | 手动部署 | GitHub Actions 自动构建 |
| 监控 | 基础日志 | Prometheus + Grafana |
| APM | 无 | Sentry 错误追踪 |
