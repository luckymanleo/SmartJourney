# SmartJourney（智旅）需求文档

> 版本：v2.0 | 日期：2026-06-08 | 状态：核心 MVP 完成，体验优化中

---

## 1. 项目概述

SmartJourney（智旅）是 AI 驱动的智能旅行规划平台。用户用自然语言描述出行需求，系统自动搜索真实机票/火车/酒店/景点/美食数据，生成完整行程方案。支持移动端 + PC Web 双版本。

### 核心差异化

| 对比 | 传统 OTA | SmartJourney |
|------|---------|--------------|
| 交互 | 表单筛选 | 自然语言 → AI 规划 |
| 编排 | 手动 | LLM 自动编排多日行程 |
| 数据 | 单一平台 | MCP 聚合飞猪数据 |
| 体验 | 多次跳转 | 一站式规划 + SSE 流式 |

## 4. 非功能需求

### 4.1 性能
- 搜索接口 < 30s（受 MCP 服务端影响）
- AI 规划 SSE 首事件 < 5s
- 前端首屏 < 1s（gzip + 缓存）
- MCP 缓存命中 < 100ms

### 4.2 安全
- JWT 鉴权 + 用户数据隔离
- 验证码 60s 限流（Redis）
- 手机号脱敏

### 4.3 可靠性
- Redis 不可用时 MemoryRedis 降级
- MCP 故障时优雅提示
- 日志文件按日切分，error 单独记录
