# SmartJourney（智旅）需求文档

> 版本：v1.1 | 日期：2026-06-05 | 状态：Phase 1 完成，Phase 2-4 部分提前实现

---

## 1. 项目概述

### 1.1 项目背景

SmartJourney（智旅）是一款面向 C 端消费者的智能出行规划平台。通过集成飞猪、美团等第三方旅行数据源（以 MCP 工具形式接入），结合 AI 大模型能力，为用户提供一站式行程查询、智能规划、比价推荐服务。

### 1.2 产品定位

**智能行程规划引擎** — 以 AI Agent 为核心，用户用自然语言描述出行需求，系统自动搜索机票/火车票/酒店/景点/美食/交通，生成最优行程方案。Phase 1 聚焦"查询+规划"，预订走第三方跳转链接。

### 1.3 目标用户

- 自由行旅客（家庭出游、情侣、独行）
- 商务出行人士
- 学生背包客
- 短期周末游用户

### 1.4 核心差异化

| 对比维度 | 传统 OTA 平台 | SmartJourney |
|----------|-------------|--------------|
| 交互方式 | 表单筛选 | 自然语言对话 + AI 规划 |
| 交通方式 | 单一查询 | 混合交通智能组合（飞机+高铁+打车） |
| 行程编排 | 手动添加 | AI 自动编排多日行程 |
| 数据源 | 单一平台 | 飞猪+美团双源聚合 |
| 个性化 | 固定筛选条件 | 学习用户偏好，主动推荐 |

## 3. 非功能需求

### 3.1 性能
- 搜索接口响应时间 < 3 秒（含 MCP 调用）
- AI 规划首次结果流式返回 < 5 秒
- 页面首屏加载 < 2 秒
- 支持 1000 并发用户

### 3.2 可用性
- 系统可用性 ≥ 99.5%
- MCP 数据源故障时优雅降级（显示缓存数据或提示）

### 3.3 安全
- 手机号脱敏存储
- 验证码防刷（同一手机号 60 秒内限 1 次）
- API 接口 JWT 鉴权
- MCP API Key 服务端存储，不暴露前端

### 3.4 可扩展
- MCP 工具热插拔（新增数据源无需改业务代码）
- 前端组件化，支持按功能模块迭代

### 3.5 移动端适配 ✅
- [x] 移动优先响应式设计（React 18 + TypeScript + Vite + TailwindCSS + Zustand）
- [x] 触摸友好交互（滑动、长按等）
- [ ] PWA 支持（离线缓存）

### 3.6 PC 端适配 ✅ 提前实现

- [x] 城市级联选择器（3434 行数据，3 级联动）

## 4. 约束与假设

### 4.1 技术约束
- 后端：Python + FastAPI
- 前端：React 18 + TypeScript + Vite + TailwindCSS + Zustand
- 数据库：PostgreSQL + Redis
- MCP 工具：飞猪旅行、美团旅行等（mako2026 系列）
- 地图：高德地图 API
- 天气：高德天气 API 或和风天气

### 4.2 业务假设
- 用户为中国大陆用户（手机号注册）
- 初期仅支持国内出行（国际出行 Phase 4+）
- 预订通过第三方跳转链接完成（Phase 1-3）
- 飞猪/美团 MCP 工具持续可用
- 高德地图 API 配额充足

### 4.3 合规
- 个人信息保护法合规（用户数据加密存储）
- 不存储第三方平台账号密码
- 预订跳转时明确告知用户离开本站
