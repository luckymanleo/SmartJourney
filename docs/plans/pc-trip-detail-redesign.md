# PC 行程详情页排版重构方案

> 日期: 2026-06-09 | 状态: ✅ 已实施

---

## 当前问题

### 1. 双重 Padding

```
LayoutPC main: p-6 (24px)
TripDetailPagePC: p-6 (24px)
→ 实际内边距 48px，内容区域被挤压
```

### 2. 行程 Tab 左右比例失衡

```
当前: Timeline 50% | Weather + Info + Budget 50%
问题: Timeline 内容多（多天行程），占的却是同样宽度
      右侧 Info 面板内容少，大量空白却占了 50%
```

### 3. 地图 Tab 左侧太窄

```
当前: 左侧行程列表 40% | 地图 60%
问题: 行程项标题长（如 "G904 深圳北→天津西（高铁）"），40% 宽度下被截断
```

### 4. Header 行拥挤

```
当前: [←返回] [标题+Tag] [Tab bar]            [🗑️]
      所有元素挤在一行，Tab bar 和标题抢空间
```

### 5. Tab 与内容距离远

Header 下方直接是内容区，Tab bar 在 Header 行右侧——用户视觉上 Tab 和内容没有关联感。

---

## 新布局

### 整体结构

```
┌────────────────────────────────────────────────────────┐
│ [← 返回行程列表]                           [🗑️ 删除]  │  ← 干净 Header
│ 标题 + RouteTag                                         │
├────────────────────────────────────────────────────────┤
│ [ 行程 ] [ 🗺️ 地图 ]                                    │  ← Tab 独立一行
├────────────────────────────────────────────────────────┤
│                                                        │
│   (Tab 内容区，占满剩余高度)                              │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 行程 Tab

```
┌───────────────────────┬──────────────────┐
│ Timeline (58%)        │ Info (42%)       │
│                       │                  │
│ 📅 Day 1              │ 🌤️ 天气          │
│  ✈️ 航班... → 预订     │ 出发地 目的地     │
│  🏨 酒店...            │                  │

│                       │ 📅 日期           │

│                       │                  │
│                       │ 💰 预算明细       │
└───────────────────────┴──────────────────┘
```

### 地图 Tab

```
┌───────────────────────┬──────────────────────────┐
│ 天选择 + 行程 (35%)    │ 地图 (65%)               │

```

## 改动清单

| # | 改动 | 说明 |
|---|------|------|
| 1 | 移除 TripDetailPagePC 外层 `p-6` | LayoutPC main 已有 padding |
| 2 | Tab 移到 Header 下方独立行 | 视觉上 Tab 管内容 |
| 3 | Header 简化：返回 + 标题 + 删除 | 一行干净 |
| 4 | 行程 Tab 比例：左 58% / 右 42% | Timeline 多给空间 |
| 5 | 地图 Tab 比例：左 35% / 右 65% | 地图最大化 |
| 6 | 信息卡间距调小 `space-y-3` | 紧凑 |
| 7 | 字体统一：标题 `text-sm`，内容 `text-[13px]` | PC 最小 14px 铁律 |
| 8 | 天选择器按钮加宽 `px-4` | 更容易点击 |

---

## 不改动

- TripMap 组件
- TripTimeline 组件
- BudgetPanel 组件
- LayoutPC
- 移动端 TripDetailPage
